# Dependency Audit

Audit date: 2026-07-16

Scope: current direct dependencies declared in `package.json`, plus completed removals. Transitive dependencies are not audited individually; they are controlled by their direct parent and `pnpm-lock.yaml`.

## Decision rule

Keep a dependency only when its maintenance and implementation cost is lower than a native alternative. Runtime dependencies deserve more scrutiny than development-only tooling. Before removing a dependency, preserve public formats and verify Chrome and Firefox builds.

## Recommended order

1. Decide whether automatic import sorting justifies the Prettier plugin.
2. Keep Svelte/WXT and their supporting development dependencies unless a separately approved vanilla-extension migration is funded and planned.

## Completed removals

| Dependency          | Change                                                                                                | Compatibility evidence                                                                                                                                                                             |
| ------------------- | ----------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `svgo` `3.3.3`      | Removed because it had no source, configuration, or script reference.                                 | `pnpm run typecheck`, Chrome build, and Firefox build passed after removal.                                                                                                                        |
| `js-base64` `3.7.8` | Replaced with `lib/utilities/base64.ts`, which uses `TextEncoder`, `TextDecoder`, `btoa`, and `atob`. | `tests/fixtures/bookmark-export-compatibility.json` contains legacy v2 and Unicode v5 strings generated by `js-base64`. `pnpm run test:bookmark-exports`, typecheck, and both browser builds pass. |

## Audit evidence

- `pnpm why` confirms every current entry above is a direct dependency of PoeTradePlus; Svelte and WXT are also required by `@wxt-dev/module-svelte`.
- Before removal, code search found `js-base64` only in bookmark export/import, 41 `lucide-static` imports, and no `svgo` source/config/script reference.
- `wxt.config.ts` registers `@wxt-dev/module-svelte`; `.prettierrc.mjs` registers the import-sorting plugin.
